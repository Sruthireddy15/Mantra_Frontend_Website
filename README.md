# Sruthi Reddy Educational Institute Website

##  Project Overview
This project is a responsive educational institute website developed using HTML, CSS, and JavaScript. It provides information about the institute, available courses, campus gallery, and student registration. The website is designed with SEO best practices and a user-friendly interface.

##  Features
- Responsive design for desktop and mobile devices
- Home, About, Courses, Gallery, and Contact pages
- Student registration form with JavaScript validation
- SEO-friendly page titles and meta descriptions
- Image alt text for accessibility
- Live chat integration using tawk.to
- Clean and modern user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Netlify (Deployment)
- GitHub (Version Control)
